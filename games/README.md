# Games

This folder contains all the game-related projects, like tictactoe, ludo, blackjack, etc. some of them are complete and some of them are incomplete. 

## Projects

- [Blackjack](./blackjack) : a simple implementation of the Blackjack card game.
- [Cookie Clicker Acceleration Time Calculator](./cookie_clicker_acceleration_time_calculator) : a script to calculate the time it takes to accelerate in the Cookie Clicker game.
- [Ludo](./ludo) : a incomplete implementation of the Ludo board game. the scope was HUGE, I wanted to make a fully configurable Ludo game with AI and everything.
- [Minesweeper Solver](./minesweeper_solver) : a script to solve Minesweeper boards. it doesn't work. it would read the board from a screenshot and then try to solve it, but it was not very good at reading the board, so I gave up on it.
- [Rust Bucket Simulator](./rust_bucket_simulator) : a script to simulate the Rust Bucket game. my plane was to find the minimum moves possible for each one of the 40 levels. but again, the scope was huge and I stopped before implementing the actual solver. I might come back in the future.
- [Simple Coin Guess Game](./simple_coin_guess_game) : a simple implementation of a coin guess game, where the player has to guess if the coin is heads or tails.
- [Super Hotting Games](./super_hotting_games) : a script that would stop the game if it detected that the player was not moving. it was based on the game Super Hot, where time only moves when the player moves. it doesn't work and it never worked. I still dream of making it work one day, but I don't know if I will ever do it.
- [Tic Tac Toe](./tic_tac_toe) : a simple implementation of the Tic Tac Toe game.
- [Uno](./uno) : I only implemented the deck :D

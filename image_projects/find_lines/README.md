# Find Lines

There are 3 implementations of line finding algorithms in this folder. the first one is a simple implementation that uses the hsv values, then we have RGB and an RGB cumulative. I don't remember how it is used, but when I come back to this project I will update the readme with better descriptions of each implementation and how to use them.

## Projects

- [Find Lines HSV](./hsv) : a script to find lines in images using the hsv values.
- [Find Lines RGB](./rgb) : a script to find lines in images using the rgb values.
- [Find Lines RGB Cumulative](./rgb_cumulative) : a script to find lines in images using the cumulative rgb values.

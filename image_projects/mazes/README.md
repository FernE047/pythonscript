# Mazes

This folder contains various projects related to mazes. I have always been fascinated by mazes and I have a lot of ideas for projects in this area.

## Projects

- [Average Multiple Mazes](./average_multiple_mazes) : takes mazes from a folder and creates an average maze from them. the resulting image is an "average" of all the mazes in the folder. it is useless
- [Color Maze Decisions](./color_maze_decisions) : it colors the path depending on how much freedom of choice you have at each point.
- [Maze Maker](./maze_maker) : it generates a maze using a random growing tree algorithm.
- [Maze Double Path Maker](./maze_double_path_maker) : it generates a maze with multiple solution paths. it uses the same logic as ```maze_maker``` but it starts from the start and the end at the same time, and at the end it connects the two paths in a random point, creating a maze with multiple solution paths.
- [Maze Solver](./maze_solver) : solves a maze.

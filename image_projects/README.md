# Image Projects

This folder contains all projects that are related to image processing, like blurring, cropping, color manipulation, etc. This is the folder I have the most projects in, because I am very interested in image processing and I have a lot of ideas for projects in this area. some code are done more as an artistic experiment than a real project. one highlight from this folder is the morph projects, where I morph two images together using interpolation, but not like other developers that use triangular linear interpolation, no. I did my own type of configurations, that involved line, areas and layers. it is pretty cool, but the results can be lackluster for no reason sometimes.

## Projects

- [Arithmetic Progression Between Images](./arithmetic_progression_between_images) : a script to create an arithmetic progression between two images.
- [Average Image](./average_lmage) : a script to calculate the average image between two images.
- [Average Image on Folder](./average_image_on_folder) : a script to calculate the average image from all images in a folder.
- [Blur Image](./blur_image) : it blurs 3 images using 3 different coordinates.
- [Cellular Automaton](./cellular_automaton) : a script to simulate a cellular automaton on an image.
- [Color Palette](./color_palette/README.md) : projects related to color palette generation and manipulation.
- [Convert Image for Better OCR](./convert_image_for_better_ocr) : a script to preprocess images for better OCR results.
- [Crop Image in Mass](./crop_image_in_mass) : a script to crop multiple images in a folder.
- [Cumulative Pixel Histogram](./cumulative_pixel_histogram) : a script to generate a cumulative pixel histogram for an image.
- [Display Image as ASCII](./display_image_as_ascii) : a script to display an image as ASCII art.
- [Display Video as ASCII](./display_video_as_ascii) : a script to display a video as ASCII art.
- [Early Image Compression Visualizer](./early_image_compression_visualizer) : a script to visualize how early image compression algorithms used to work. it squish the image by removing repeated pixels from current row/columns.
- [Earworms](./earworms/README.md) : this projects takes a music lyrics as a vector and creates a repetition matrix where it highlights the repeated parts of the lyrics. it is a pretty cool way to visualize earworms in music. it's based on [Vox pop video](https://www.youtube.com/watch?v=HzzmqUoQobc) about earworms. I thought it was a very artistic project, so I made my own implementation to play with it :D
- [Extract Level Map from Video](./extract_level_map_from_video) : a script to extract the level map from a video of a game. it never worked very well, because modern games use parallax scrolling and other techniques that make it hard to extract the level map. this is one of my dream projects. I love seeing level maps made by players specially speedrunners.
- [Find Lines](./find_lines/README.md) : scripts to find lines in images. better descriptions on the readme inside the folder.
- [Generate 3bit RGB Image](./generate_3bit_rgb_image) : a script to generate a 3bit RGB image from a normal image. on 3bit RGB images, each color channel can only have 2 values, 0 and 1. so the resulting image only have 8 possible colors. A R T I S T I C 🌟
- [Generate a Blur](./generate_a_blur) : a script that just make a specific 11x11 blur. it takes way too long right now.
- [Generate BW Illusion](./generate_bw_illusion) : a script to generate a black and white illusion image. it is based on a video I have watched only once on youtube, it says that if you insert some black and white lines in the image, and keep the rest of the image the same, it creates an illusion that the whole image is colorful. it's silly
- [Generate HOG](./generate_hog) : a script to generate a Histogram of Oriented Gradients (HOG) visualization for an image.
- [Generate Image RGB Less Bits](./generate_image_rgb_less_bits) : a script to generate an image with less bits for each color channel. it is similar to the 3bit RGB image, but it generates all the possibilities. V E R Y A R T I S T I C 🌟
- [Generate Random Island Map](./generate_random_island_map) : a script to generate a random island map.
- [Generate RGB Color Space as Image](./generate_rgb_color_space_as_image) : a script to generate a folder image of the RGB color space where each image is a layer slice from 0 to 255. we generate 3 times, one for each color channel.
- [Hue Recolor from Folder](./hue_recolor_from_folder) : a script to recolor an image with all the hues offset by a certain amount, and generate one image for each hue offset.
- [JPG Compression Visualizer](./jpg_compression_visualizer) : a script to visualize JPG compression to extreme levels by repeatedly saving the image as JPG with low quality and then opening it again. it doesn't work because I didn't knew you have to set the saving quality to low every time you save the image, so it just saves the image with the default quality every time, which is VERY HIGH.
- [Mazes](./mazes/README.md) : various projects related to mazes. better descriptions on the readme inside the folder.
- [Morph Projects](./morph_projects/README.md) : various projects that morph two images together using interpolation. better descriptions on the readme inside the folder. my fav project
- [Paste All Images from Folder](./paste_all_images_from_folder) : a script to paste all images from a folder into one image. it pastes the images considering the center point. this code3 exists because I wanted to paste all pokemon sprites and see the shape of the resulting pokemon.
- [Radial Unwrap](./radial_unwrap) : a script to unwrap an image radially. it takes an image and unwraps it from the center to the edges. this script is awesome to use on faces where the nose is the center, it creates a very cool effect. I have used it on Luigi (mario's brother) and it's hilarious. you gotta try it.
- [Random Decreasing Spiral](./random_decreasing_spiral) : a script to generate a random decreasing spiral pattern. i Wa bored :v
- [Remove Sprite Background](./remove_sprite_background) : a script to remove the background from a sprite.
- [Separate RGB Channels](./separate_rgb_channels) : a script to separate the RGB channels of an image into three different images. Red, Green and Blue.
- [Shuffle Image Pixels](./shuffle_image_pixels) : a script to shuffle the pixels of an image randomly.
- [Sprites Heat Map](./sprites_heat_map) : a script to generate a heat map of the sprites in a folder. so you can see which areas are more common in the sprites.
- [Threshold Image](./threshold_image) : a script to threshold an image. I don't remember the details of how it works.
- [Turn Image into Nail Art](./turn_image_into_nail_art) : a script to turn an image into a circular nail art pattern. it's based on a youtube video I watched a long time ago. I remember it didn't worked well, and the video was not very clear on how to do it, so I had to improvise a lot.
- [Turn Image into Square](./turn_image_into_square) : a script to turn an image into a square. imagine kirby... but square.

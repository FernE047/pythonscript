# Color Palette

This folder contains projects related to color palette generation and manipulation. this is purely artistic, it's similar to applying a filter on an image.

## Projects

- [Apply Palette to Folder](./apply_palette_to_folder) : a script to apply a color palette to all images in a folder.
- [Extract Palettes from Folder](./extract_palettes_from_folder) : a script to extract color palettes from all images in a folder.
- [Extract Palettes from Image](./extract_palettes_from_image) : a script to extract color palettes from an image.
- [Generate BW Palette](./generate_bw_palette) : a script to generate a black and white color palette from an image.

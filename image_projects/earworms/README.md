# Earworms

This project takes a music lyrics as a vector and creates a repetition matrix where it highlights the repeated parts of the lyrics. it is a pretty cool way to visualize earworms in music. it's based on [Vox pop video](https://www.youtube.com/watch?v=HzzmqUoQobc) about earworms. there are currently 3 codes in this folder. the standard one, a circular implementation and a heatmap analyzes of the first two implementations.

## Projects

- [Standard Earworms](./standard) : the standard implementation of the earworms project. it creates a repetition matrix where it highlights the repeated parts of the lyrics.
- [Circular Earworms](./circle) : a circular implementation of the earworms project.
- [Earworms Heatmap](./generate_heatmap) : a script to generate a heatmap of the repetition matrix created by the standard and circular implementations.

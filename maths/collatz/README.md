# Collatz Projects

Some explorations related to the Collatz conjecture. I have been exploring a way to compress the conjecture function using a transformation that generates collatz with different levels. I still need to regularizez though.

## Projects

- [Collatz level 0](./calculate_collatz_O) : this is a implementation of the Collatz conjecture using the original function. it is the standard implementation of the Collatz conjecture.
- [Collatz level 1](./calculate_collatz_1) : this is a implementation of the Collatz conjecture using the first level of the transformation.
- [Collatz level 2](./calculate_collatz_2) : this is a implementation of the Collatz conjecture using the second level of the transformation.
- [Collatz level 0, 1, and 2](./calculate_collatz_O_1_and_2) : this is a implementation of the Collatz conjecture where you can choose which level of the transformation to use.
- [Concatenation Collatz](./concatenation_collatz) : this is a implementation of the Collatz conjecture using concatenation of numbers.
- [Display Collatz sequence 0, 1, and 2](./display_collatz_sequence_0_1_and_2) : this is a implementation of the Collatz conjecture where you can display the sequence for different levels of the transformation.
- [Generate new Collatz](./generate_new_collatz) : this is the transformation that generates new Collatz functions with different levels.
- [Plot Collatz 0 and 2](./plot_collatz_0_and_2) : this is a implementation of the Collatz conjecture where you can plot (using matplotlib) the sequence for different levels of the transformation. it is interesting to see how the sequence changes with different levels of the transformation.

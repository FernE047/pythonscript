# Maths

I have always been fascinated by mathematics. this folder contains various projects related to mathematics, like number theory, algebra, calculus, etc. collatz is a highlight of this folder.

## Projects

- [Appended Repdigit Prime Factorization](./appended_repdigit_prime_factorization) : a script where we take appended repdigits (like 311, 3111, 31111, etc) and we factor them into their prime factors.
- [Approximate Khinchin Constant](./approximate_kinchin_constant) : we approximate the Khinchin constant by calculating the geometric mean of the first n terms of the continued fraction expansion of a random number.
- [Collatz](./collatz/README.md) : multiple projects related to the Collatz conjecture. my fav math problem :emoji_heart_eyes:
- [Complex Exponential Applied to Images](./complex_exponential_applied_to_images) : we take an image and we apply the complex exponential function to each pixel value. the resulting image is artistic.
- [Concatenation](./concatenation/README.md) : multiple projects related to concatenation of numbers.
- [Convert Continued Fractions](./convert_continued_fractions) : convert a number to its continued fraction representation and vice versa.
- [Convert to Irrational Bases](./convert_to_rational_bases) : convert a number to irrational bases like pi. the folder has a typo it seems.
- [Euler Squares Solver](./euler_squares_solver) : solves the euler squares problem for a given n.
- [Factorial](./factorial) : calculates the factorial of a given number.
- [Generate Simple Fractal Matrix](./generate_simple_fractal_matrix) : generates a simple fractal matrix.
- [Project 4D onto 2D](./project_4d_onto_2d) : projects a 4D object onto a 2D plane.
- [Rubik Cube Chains](./rubik_cube_chains) : explores chains of moves on a Rubik's Cube.
- [Self Replicating Fractal](./self_replicating_fractal) : generates a self-replicating fractal.
- [Solve by Bisection](./solve_by_bissection) : solves equations using the bisection method.

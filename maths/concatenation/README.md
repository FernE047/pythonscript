# Concatenation

Some explorations related to concatenation of numbers.

## Projects

- [Concatenation Associative Property](./check_concatenation_associative_property) : a script to check if concatenation of numbers is associative. it is not, but it is interesting to see the results.
- [Multiplication Equal Concatenation](./check_multiplication_equal_concatenation) : a script to check if multiplication of numbers is equal to their concatenation. it is not, but it is interesting to see the results.
- [Find Cycles Sum Concatenation](./find_cycles_sum_concatenation) : a script to find cycles in the sum of concatenated-cropped numbers.
- [Last Fermat Theorem Concatenation](./last_fermat_theorem_concatenation) : a script to explore the last Fermat theorem using concatenation of numbers.

# Artificial Intelligence

This folder contains projects related to artificial intelligence, including puzzle solvers, game AIs, and some Markov chains.

## Projects

- [Horse Chess Solver](./horse_chess_solver): A solver for the horse chess problem. there is a [multidimensional version](./horse_chess_solver/multi_dimensional) and a [2d only version](./horse_chess_solver/only_2d).
- [Markov Text Generators](./markov_text_generators/README.md): Text generators using Markov chains. 4 projects very similar to each other, but with different implementations and different results.
- [Nonogram Solver](./nonogram_solver): A solver for nonogram puzzles.
- [Sudoku Solver](./sudoku_solver): A solver for Sudoku puzzles. there's a SUPER fast version without UI and a slower version with a text-based UI. I basically just wanted to see how fast I could make a sudoku solver.
- [Truco AI](./truco_ai): An very simple AI for the BRAZILIAN 🇧🇷🥳🇧🇷 card game Truco.
- [TSP Solver](./tsp_solver): A solver for the Traveling Salesman Problem. two versions, one [using classes](./tsp_solver/using_classes) and one [without classes](./tsp_solver/no_classes).

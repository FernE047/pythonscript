# Markov by Letters

this is one of my markov text generators, it generates text based on the letters of the input text, the input text are messages copied and pasted from a whatsapp conversation, the generated messages are printed on the screen, by default it print 1000. the markov chain used is based on one previous letter.

## Usage

1. rename the file `in/whatsapp_messages.txt.example` to `in/whatsapp_messages.txt`
2. copy and paste the messages you want to use as input in the file `in/whatsapp_messages.txt`
3. run the file `main.py` to clean the input messages and save them in the file `clean_input.txt`

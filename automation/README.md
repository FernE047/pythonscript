# Automation

This folder contains scripts that automate some tasks, like copying a pdf over and over, or trying to automate old games gameplay. they are all pretty useless now because they use pyautogui with hardcoded coordinates, so they only used to work on my old computer that I don't have anymore

## Projects

- [Auto Clicker](./auto_clicker) : a simple auto clicker that clicks at a certain interval. it is not very useful, but it was fun to do at the time.
- [Automate Copying Graphs Book](./automate_copying_graphs_book) : a script to automate copying graphs from a book.
- [Automate PDF File Creation](./automate_pdf_file_creation) : a script to automate the creation of PDF files.
- [Display Mouse Variables](./display_mouse_variables) : a script to display mouse variables. VERY useful. based on ```automate the boring stuff``` by Al Sweigart.
- [Games](./games/README.md) : three game-related projects. including Sonic, Mario and Pokemon. all incomplete.

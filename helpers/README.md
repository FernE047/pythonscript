# Helpers

the scripts on this folder were made to automate boring tasks that I used to do, like converting captions to a format that I could use on my videos, calculating complementary activities for my classes, converting word files to pdf, etc. they are not really projects and are only useful for the specific tasks they were made for.

## Projects

- [Auto Indent](./auto_indent) : a script to auto indent python files.
- [CC Caption Converter](./cc_caption_converter) : a script to convert closed captions files to a format that I could use on my videos.
- [Complementary Activities Calculator](./complementary_activities_calculator) : a script to calculate the complementary activities that I needed to do for my classes.
- [Convert Word to PDF](./convert_word_to_pdf) : a script to convert word files to pdf.
- [Learning Helper](./learning_helper) : a script to help me learn new things.
- [Library System](./library_system) : a script to manage a library system.

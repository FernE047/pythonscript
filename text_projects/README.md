# Text Projects

Projects related to text manipulation, generation, and ocr. they use a library called pytesseract to do OCR.

## Projects

- [Generate Names](./generate_names) : a script to generate random names. based on letter frequency.
- [Get a Random Fanfic](./get_a_random_fanfic) : a script to get a random fanfic. it uses spirit fanfiction.
- [Japanese OCR](./Japanese_ocr) : a script to perform OCR on Japanese images. because I was playing a lot of Japanese games and I wanted to understand what they were saying. 
- [Simple OCR](./simple_ocr) : it just does OCR on an image and prints the result. it is a very simple implementation
- [Better OCR](./better_ocr) : it does OCR on an image, but it also preprocesses the image to improve the OCR results.
- [OCR from Folder](./ocr_folder) : a script to perform OCR on all images in a folder.
- [Simple OCR from Folder](./ocr_folder_simple) : a simple implementation of OCR on all images in a folder.
- [OCR from Folder Using Threshold](./ocr_folder_using_threshold) : performs OCR on all images in a folder using a thresholding technique.

# Brainf___

there is only 3 projects in this folder, two interpreters for the Brainf___ esoteric programming language, and a script to generate brainf___ code ad infinitum (só que não).

## Projects

- [Brainf___ Infinite Tape](./brainf___infinite_tape) : this interpreter implements the Brainf___ language with an tape that can reach infinite values.
- [Brainf___ Roll Over](./brainf___roll_over) : this interpreter implements the Brainf___ language with a tape that can only reach values between 0 and 255. if it reaches 256, it rolls over to 0.
- [Generate Brainf___ Code](./generate_brainf___scripts) : a script to generate Brainf___ code.

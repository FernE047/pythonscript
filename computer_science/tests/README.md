# Tests

This folder contains scripts that I made just to test functionality of some code or library. they are not really projects, but they are useful for testing and learning.

## Projects

- [Test Arg Parse](./test_arg_parse) : a script to test the argparse library.
- [Test Classes](./test_classes) : a script to test classes in python.
- [Test Google Search](./test_google_search) : a script to test how to do a google search in python.
- [Test ImageDraw Line](./test_imagedraw_line) : a script to test the ImageDraw line function from the PIL library.
- [Test Lambda](./test_lambda) : a script to test lambda functions in python.
- [Test Markovify](./test_markovify) : a script to test the markovify library.
- [Test PyPDN](./test_pypdn) : a script to test the PyPDN library, which is a library to create and manipulate .pdn files. I LOVE Paint.net. this software have my heart ❤❤❤
- [Test Read Write Binary](./test_read_write_binary) : a script to test reading and writing binary files in python.
- [Test Read Write File](./test_read_write_file) : a script to test reading and writing text files in python.
- [Test Sys Exit](./test_sys_exit) : a script to test the sys.exit function in python.

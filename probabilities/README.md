# Probabilities

This folder contains all projects related to probabilities and statistics. I have a bunch of scripts to counting and simulating probabilities.

## Projects

- [Advantage Dice Probabilities](./advantage_dice_probabilities) : a script to calculate the probabilities of rolling a certain number with advantage dice. it calculates the probabilities for both normal advantage and disadvantage.
- [Android Password Grid Counter](./android_password_grid_counter) : a script to count the number of possible passwords on an Android lock screen grid.
- [Dice Probabilities Days Display](./dice_probabilities_days_display) : a script to display the probabilities of rolling certain numbers on dice over a number of days.
- [Ludo Probabilities](./ludo_probabilities) : a script to calculate the probabilities of different outcomes in the game of Ludo. it's incomplete
- [Memory Games Probabilities](./memory_games_probabilities) : a script to calculate the probabilities of success in THE memory games.
- [Pokedex Census](./pokedex_census) : a script to register and display pokemon informations like a population census.
- [Pseudo Number Generator](./pseudo_number_generator) : a script to display probabilties of pseudo-random numbers.
- [Slow Distribution](./slow_distribution) : a script that makes a slow distribution.
